透過 adb 把 android 畫面顯示到 PC 上

asm.jar 全名 Android Screen Monitor， 一款搭配 adb 就能把 android 畫面顯示在桌機上的軟體。 不用裝 Android Studio，只要有 adb 就可以。

2017 年 CCNS 辦了個 QA 問答大賽宣傳， 有活動是在社博上來我們攤位答題，現場答對有獎品。 主要是要把答題畫面展示出來，才會吸引人。 所以需要將畫面顯示出來的功能。

Android Debug Bridge

adb 可以用 windows 下那種很陽春在 cmd 裡跑的， 也能用 linux 下的。 反正 adb devices 有抓到裝置就好。

我是習慣用 wifi 不用線，所以裝了 adb over wifi， 所以 adb 是顯示 IP 和 port 5555 10.42.0.181:5555。 如果你要用可以看我 介紹的文章 。

Android Screen Monitor

google 應該就有下載點，但我還是 自己鏡像 一次。 github 寫已經停止更新 ， 要大家改用新的 vysor ， 但 vysor 還要裝 apk，而且要用 eletron 裝桌面版感覺很麻煩。 chrome app 版是用 nacl 寫的，chromium 不能用， 而且 nacl google 不是要放棄了嗎。

操作影片

用 vlc 錄的 ： https://youtu.be/d0Fo-AnCRqU

  1. 插 usb 線連上電腦，並開啟偵錯模式。
  2. 用 adb devices 確認已連線。
  3. java -jar asm.jar 開啟螢幕。
