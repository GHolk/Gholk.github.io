rtx csrs ppp 比較

rtx 是星基即時動態定位，可望成為取代 vbs-rtk 的下一代產品。 除了 vbs-rtk 的定位精度，更免除了虛擬基站的計算負擔。 本文將 rtx 與 csrs ppp 比較，探討 rtx 的定位精度。

isrs summary

RTX and CSRS PPP kinematic positioning's accuracy study

This study inspect the accuracy of RTX by comparing RTX ™ to the CSRS PPP service. RTX is a realtime kinematic satelite base positioning service, which claim that its positioning is as precise as traditional VBS - RTK . Traditional VBS-RTK is constrained by the server's computing ablitity because the server need to compute a virtual base station for each user. Howevery, satelite base positioning service like RTK will broadcast the correction signal to every user; same correction signal can apply to all users, so satelite base positioning service only need one times computing. It is expect that satelite base RTK will replace traditional VBS-RTK.

綜合比較

綜合來說，rtx 較 csrs online 的解算成果較接近公告作標值， 但 csrs online 的精度較佳。 而本地端的舊版本 csrs 精度和準度 都明顯低於 rtx 與 csrs online，就不作統計。

結論

大致上 rtx csrs 的定位分布都接近常態分布， csrs ppp 的精度較佳，而 rtx 的精度稍低， 但平均較接近公告座標。
