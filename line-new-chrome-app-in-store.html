line 換新 chrome app

因為用 linux，不能裝 chrome 桌面版， 只能裝 chromium 再用 chrome store 裡的 line app 將就。 最近 line 啟動，會警告舊版會停止支援。 本來新版裝不起來，發現是 chromium 要更新， 跟上 chrome 版本，就能裝了， 但不像舊 line chrome app 能獨立啟動。

之前的 chrome app 可以獨立啟動， chromium --app-id=menkifleemblimdogmoihpfopnplikde 直接啟動， 試過把 app-id 改成新版的不行。 但現在新版好像不行，一定要開 chromium， 再從工具列裡開。 雖然之後就能把 chromium 關了，只留 line。

還感覺不出二者有什麼差， 不太懂為什麼要更新。 只是新版多了邊框，比較像一般視窗風格； 而且新版可以自由調整視窗大小， 舊版和之前 discord 一樣，有限最小寬度， 不能調到半個螢幕寬。 之前舊版會一些功能不能用，像投票， 不知道新版有沒有改善。
