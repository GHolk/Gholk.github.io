x11vnc 圖形介面啟動

x11vnc 是一個在 x window 既有架構下， 提供 vnc 連接的程式。 除了命令列介面，也有簡單的圖形介面可以用。 缺點是一般是用來分享已登入的 x window， 要已經有登入的 x window， 而且該 x window 要一直開著不能關掉。

我一開始在找遠程登入時， 發現我系統裡不知道為什麼有 x11vnc 這個指令， 研究了一下，就會用密碼登入了。 可以用 x11vnc -usepw 就能把密碼存在 ~/.vnc/passwd 。

後來發現他有一個 launcher， 啟動會帶有圖形介面，能手動設定密碼和其它雜七雜八選項。 執行前會要你設定 port 和要不要 ssl， 然後才正式執行，並會常駐 icon 在通知欄。

之後會跳一個設定介面，預設是沒有密碼，也不允許登入， 要勾選允許登入，才能登入。 可以不設密碼，但就比較危險。 但有另一個選項能手動允許登入， 就會變成有人嘗試登入時， 會跳出視窗要你允許或拒絕； 就能不設密碼，一律手動審核。

也有選項可以讓登入者只能觀看不能操作， 就單純分享畫面用。
