載入 disqus

由於 blog 是 github page，就用 disqus 當留言系統。 之前在想要怎麼載入，因為 disqus 蠻大的， 我部落格的文章載入很快，但 disqus 就超慢。 最後決定還是直接載入。

用按鈕載入

之前用 firefox 看，發現 disqus 載入超多東西的， 我原本 blog 只有三四個 css 和 js， disqus 載入了 30 幾個，加起來快 1MB。 就有點不爽，載入那麼多幹麻。 所以就寫成一個按鈕，按了才會載入 disqus，節省流量。

改回來直接載入

後來發現一個盲點，因為我的 blog 沒什麼人留言， 要是每次點開都發現沒人， 那久而久之就不會有人想點開了。 只好又改回直接載入。
