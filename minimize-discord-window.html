調整 discord 視窗大小

discord 視窗忽然不能調小於某一寬度， 便 google 到 raddit 上有人抱怨， discord 團隊就迅速出來滅火道歉，並提供解法。 在 ~/.config/discord/settings.json 加入 "MIN_WIDTH":0, "MIN_HEIGHT":0 即可。

主要是一些視窗介面都有自動調視窗大小的功能， 拖曳視窗到滑鼠碰到螢幕左或右邊界為止， 視窗就會調成半個螢幕寬一個螢幕高， 右上或左下則會變成半高半寬。 就能在剩下的螢幕做事， 還能看到原本的視窗。

以前習慣有事會把 discord 拖到側邊或側上， 昨天 discord 忽然不能這樣調，覺得不方便。 就 google 找解法。 看到 raddit 上有串 ， discord 團隊表示是因為縮太小排版會亂掉， 所以有設最小高寬。

raddit 就一片罵聲，說亂掉是我的事， 你不應該限制我怎麼調整視窗。 那個開發者也有提供解法， 在 settings.json 裡設 MIN_HEIGHT MIN_WIDTH 為 0。 總之看來之後會解除，或出現適合視窗極小時的排版。

{
  "IS_MAXIMIZED": false,
  "IS_MINIMIZED": false,
  "MIN_HEIGHT": 0,
  "MIN_WIDTH": 0,
  "trayBalloonShown": true,
  "WINDOW_BOUNDS": {
    "x": 344,
    "y": 29,
    "width": 940,
    "height": 736
  }
}
