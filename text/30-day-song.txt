## 名字裡有顏色的歌：[紅色高跟鞋](https://youtu.be/SaNqXIFS99k)

好像原本是中國電視劇的歌。
就覺得很好聽，有點節奏沒在拍子上的感覺。

蔡健雅的，她的歌有時會出現這種感覺，
有點隨性，放縱。


## 名字裡有數字的歌：[最後的 831](https://youtu.be/ILVlVWle8m0)

831 的歌，唱得是暑假最後一天的心聲。
大學生的最後一個暑假剛結束，
但我如果沒考上研究所，這就是最後一個暑假了吧。

我辜負了過去的自己嗎？
過去的我也沒什麼期待，可能只有仍沒交女友這個問題吧。
雖然最後的結果和當初差得遠了，
（根本沒想到會唸測量系。）也沒想過會碰 linux。

但做了就是做了，是無法改變的，
只能繼續走下去了，就這樣吧。
反正過去的我也無法來到現在來後悔了。


## 會讓你想起夏天的歌：[浪花一朵朵](https://youtu.be/beWjTKhY_l8)

這首好像和上面那首對調也可以，昨天就在這二首間猶豫。
今天本來也在想要選 **熱血** 還是真得有 **夏天** 的，
後來想還是選有夏天的好了，搞不好後面還有熱血。

其實這首不怎麼熱血，算很平凡溫暖的歌，
只是她的場景剛好在海邊，海邊就想到夏天。
mv 不知道在演三小，可能又是某部偶像劇吧。
講起來歌詞也是有點沙文主義，
把女生當成必須被守護的；管她去死。

## 會讓你想起寧願忘記的人的歌：[很幼稚嗎](https://youtu.be/nSmcSLsJSKo)

想忘記的人沒有，不想忘記的倒有一個；
那麼美好的回憶，幹麻忘掉呢？
那時候有過幾首，不過後來把思緒理清楚後，
有幾首就發現情緒不對了。

妳離開了，我依然在這裡，想著妳說過的話，
以妳的邏輯思考，用妳的方式生活；
好像能有什麼意義。
得不到的總是最美好吧。

還有考慮聽堂妹唱五月天的 *知足* ，覺得蠻不錯的。
可惜我沒能感性到唱到哭，只是個自以為的平凡人而已。
另一首也覺得不錯是 *那女孩對我說* ，
雖然很感動，但情境好像不太合，就沒有選她。
或 *很幼稚嗎* 也不錯，但不會讓人唱到想哭。

## 讓你想盡情跳舞的歌：[第一支舞](https://youtu.be/0OQ8KlAKa7k)

## 適合開車時聽的歌：[晚安台灣](https://youtu.be/Ck0xK0h_Hu8)
濁水溪公社的，不是滅火器的。
我覺得滅火器大部份的歌都不好聽，不知道在紅什麼。

在 mv 一開始的旁白就有寫，
這是小柯在某天晚上開在高速公路上，想著這麼晚了，
還是有很多人開著車，為生計為台灣經濟奮鬥。
所以想寫一首能安慰人，能提起精神，愛這塊土地的歌。

這首歌也的確是這樣，聽起來十分舒服，
能讓人對未來充滿希望，讓人好好休息。
每天都是新的一天！

## 使你開始思考生活的歌：[芝麻](https://youtu.be/P3uAPhmt_7E)

<script>
void function loadLazyIframe() {
    const youtubeAnchor = document.querySelectorAll('h2 a')
    Array.from(youtubeAnchor).forEach((anchor) => {
        const embedUrl = youtubeEmbedUrl(anchor.href)
        const lazyLoadAnchor = createLazyAnchor('open iframe', embedUrl)
        const h2 = anchor.parentNode
        h2.parentNode.insertBefore(
            lazyLoadAnchor,
            h2.nextSibling
        )
    })
    
    function youtubeEmbedUrl(short) {
        const id = short.replace(/^.*\//, '')
        return `https://www.youtube.com/embed/${id}`
    }
    function createLazyAnchor(text, embedUrl) {
        const anchor = document.createElement('a')
        anchor.href = embedUrl
        anchor.textContent = text
        anchor.addEventListener(
            'click',
            function whenLazyAnchorClick(clickAnchor) {
                clickAnchor.preventDefault()
                const iframe = document.createElement('iframe')
                iframe.src = youtubeEmbedUrl(this.href)
                iframe.allowFullScreen = true
                this.parentNode.insertBefore(iframe, this.nextSibling)
            },
            {once: true}
        )
        return anchor
    }
}()
